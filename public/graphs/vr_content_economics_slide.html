AI 3D Asset Generation
  • Meshy — text/image → 3D model in minutes. Human cell, DNA helix, or volcano from a text prompt.
  • Luma AI — real object photos → 3D NeRF model. Scan a real classroom object, place it in VR.
  • Shap-E OpenAI — open-source 3D generation. Free to run, unlimited models.
  • Cost: ₹0–500 per model vs ₹2,000–5,000 for a 3D artist.
🔁
Reuse 2D Animation Assets
  • Edastra's existing 2D animation pipeline produces sprites, textures, and character rigs that import directly into Unity as billboards or sprites in 3D space.
  • A "hybrid VR" approach — AI-generated 3D environment + 2D characters from existing pipeline. Looks great, costs 2D.
  • First 50 VR modules can be built entirely from existing 2D content — zero new asset cost.
⚡
A-Frame / Three.js over Unity
  • Unity developer = ₹1–1.5L/mo. A-Frame/Three.js web developer = ₹60–80K/mo. Same VR output, 40% lower salary.
  • WebXR with A-Frame uses HTML & JS — your existing frontend developers can build VR modules. No game engine expertise needed.
  • Open source, no royalty fees vs Unity's revenue share above $200K/yr.
